# ntumun-website
A sample for a better NTUMUN website. Currently hosted at ntumun.solderneer.me

## How to run
### Running locally
* Yarn
* Parcel

```
yarn serve
```

### Running with docker
* Docker
* docker-compose

```
docker-compose up --build 
```

## Building locally
* Yarn
* Parcel

```
yarn build
```
