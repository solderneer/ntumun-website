# ntumun-website
A sample for a better NTUMUN website. Currently hosted [here](https://ntumun.solderneer.me)

![Preview Image](/preview.png)

## How to run
### Running locally
* Yarn
* Parcel

```
yarn serve
```

### Running with docker
* Docker
* docker-compose

```
docker-compose up --build 
```

## Building locally
* Yarn
* Parcel

```
yarn build
```
